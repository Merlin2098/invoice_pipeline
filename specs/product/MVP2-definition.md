# MVP Definition

The MVP is considered complete when a user can:

1. Open the web application.
2. Upload PDF invoices.
3. Trigger the existing OCR + AI pipeline.
4. Query processed invoice data using natural language.
5. Receive business-friendly responses generated through Bedrock and Athena.

This keeps the scope focused while showcasing a complete **Serverless Data Product on AWS** that combines:

* Data Engineering
* OCR Processing
* Lakehouse Architecture
* GenAI
* Conversational Analytics
* Infrastructure as Code (Terraform)

which is significantly stronger as a portfolio project than a standalone ETL pipeline.
